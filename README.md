# Minahil Portfolio

Welcome to my personal portfolio!  
I am **Minahil**, a passionate **computer science student and web developer**.

This portfolio showcases my skills, experience, and contact details in a clean and interactive layout.

## Features

- **Home:** Intro with photo, short bio, and social links (GitHub, LinkedIn, Twitter).  
- **About Me:** Detailed description of my background, education, and interests.  
- **Skills:** Key technical skills like HTML, CSS, JavaScript, React, Node.js, SQL, and Git/GitHub.  
- **Contact:** Email and phone for easy communication.  
- **Downloadable CV:** Click the button to download my CV instantly.  
- **Responsive & Interactive:** Works smoothly on desktop and mobile, with interactive modules for About, Skills, and Contact.

## Purpose

This portfolio is designed to present my abilities professionally and make it easy for potential clients, employers, or collaborators to know more about me and reach out.
